# Game 3: 空気読みグループチャット画面

**パス:** `/games/group-chat`

**前画面:** `/games/helpdesk`（Game 2） → **現在:** `/games/group-chat` → **次画面:** `/result`（Loading画面）

**表示内容:**

- **チュートリアルオーバーレイ:** 「指示‼️」タイトル＋ゲーム説明テキスト＋「タップして開始」（半透明黒背景の全画面オーバーレイ）
- **ステージカットイン:** 各ステージ開始前に「場面N」+「Nth Situation!」を全画面表示（1.2秒間）
- **スマホ風チャットフレーム:** LINE 風グループチャット UI（`max-w-sm` / 角丸 / 枠線でスマホ端末を模擬）
  - **ヘッダー:** グループ名「第3営業部(5)」＋ オンラインインジケータ（緑丸）＋ ステージ進捗バッジ（例: `1/5`）
  - **チャットエリア（水色背景 `bg-sky-100`）:** DAY ラベル → Bot メッセージ（左寄せ・色付きアバター＋名前）→ 自分のメッセージ（右寄せ・緑色吹き出し）。選択肢表示時に最下部へ自動スクロール
  - **入力中インジケータ:** 「○○が返信中」テキスト＋バウンスドット（ステージ 3: 先輩, ステージ 5: 同期 で出現。選択肢表示中も継続表示）
- **制限時間ゲージ:** アクションエリア上部のプログレスバー（残り時間で緑→黄→赤に変化）
- **アクションエリア（`bg-amber-50`）:** 絵文字付き 4 択ボタン（縦並び）。ステージ 2 は絵文字のみ横4列グリッド

**Bot キャラクター:**

| ID        | 名前 | アバター | 色              |
| --------- | ---- | -------- | --------------- |
| boss      | 部長 | 部       | 赤 (`bg-red`)   |
| senpai    | 先輩 | 先       | 青 (`bg-blue`)  |
| colleague | 同期 | 同       | 緑 (`bg-green`) |

**ユーザー行動フロー:**

1. チュートリアルオーバーレイ（「指示‼️」）でゲーム説明を読み、タップして開始
2. **ステージカットイン**が 1.2 秒表示される（例: 「場面1 / First Situation!」）
3. チャットエリアに Bot のメッセージが順次表示される（800ms 間隔）
4. 全メッセージ表示後（ステージ 3, 5 では「○○が返信中」を 1.5 秒表示後）、4 択の選択肢が表示される
5. 制限時間（10 秒）以内に選択肢を選ぶ。時間切れの場合は無回答扱いで次ステージへ
6. ステップ 2〜5 を 5 ステージ繰り返す
7. 全 5 ステージ完了後、ゲーム終了。`/result` へ自動遷移
