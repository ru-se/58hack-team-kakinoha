# Game 2: カスタマーサポートチャット画面

**パス:** `/games/helpdesk`

**前画面:** `/diagnosis`（簡易診断）等 → **現在:** `/games/helpdesk` → **次画面:** `/games/group-chat`

**表示内容:**

- **ヘッダー:** 「【トラブル】ログイン後、ダッシュボードが表示されません」という解決すべき課題を常時表示
- **チャットタイムライン:** サポート担当（ぽんこつな人間の担当者）とユーザーの会話履歴
- **録音インジケータ（音声モード時）:** マイクアイコン、「録音中... 残り○秒」のカウントダウン、声の大きさに合わせた波形ビジュアライザー（仮実装では簡易表示）
- **入力エリア:** 音声モード（デフォルト） or テキスト入力（テキストエリア + 送信ボタン）
- **代替手段ボタン:** 画面左下に小さめで「テキスト入力に切り替え」ボタン

**ユーザー行動フロー:**

1. サポート担当の初回発言（「どうされましたか？」系）が表示される
2. デフォルトでは自動で録音開始。マイクに向かって状況を説明したり反論したりする（最大20秒で1ラリー終了）
3. 必要に応じて左下のボタンからテキスト入力に切り替え可能（テキスト時は時間制限なし、送信で1ラリー終了）
4. サポート担当の返答 → ユーザー応答を **3ラリー** 繰り返す
5. 3ラリー完了後、ゲーム終了

**APIトリガー:**

| トリガー               | API                       | タイミング                                |
| ---------------------- | ------------------------- | ----------------------------------------- |
| サポート担当の返答取得 | `POST /api/voice/respond` | ユーザーの1ラリー完了後（次の返答表示時） |
| ゲーム終了             | `POST /api/games/submit`  | ゲームが終了した瞬間（3ラリー完了後）     |

- 詳細なデータ定義は `docs/games/game2-helpdesk.md` を参照
- `POST /api/voice/respond` は演出用。現在はハードコード応答で代替

**完了時の動作:**

- Game 3 画面（`/games/group-chat`）へ自動遷移
