# 簡易性格診断画面（ベースライン取得 + ユーザー登録）

**パス:** `/diagnosis`（Step 2）

**表示内容:**

- 5つの質問（Q1〜Q5）を順番に表示
- 選択肢は4択（A, B, C, D）
- 進捗インジケーター（1/5, 2/5, ...）
- 質問内容は未定（プレースホルダー表示）

**APIトリガー:**

- 5つの質問すべてに回答完了時 → `POST /api/register`

**リクエストパラメータ:**

```json
{
  "mbti": "ENTP",
  "baseline_answers": {
    "q1_caution": "A",
    "q2_calmness": "B",
    "q3_logic": "C",
    "q4_cooperativeness": "C",
    "q5_positivity": "C"
  }
}
```

> FE側は選択肢の文字列（A/B/C/D）をそのまま送信する。
> スコア計算はバックエンド側で実行される。

**Loading表示:**

- API送信中はスピナーを表示

**エラー表示:**

- APIエラーの場合、「データ送信に失敗しました」メッセージを表示
- 「リトライ」ボタンを表示（再送信）

**成功時の動作:**

- `user_id` をlocalStorageに保存
- ゲーム開始の案内メッセージを表示
- Game 1画面（`/games/terms`）へ自動遷移
