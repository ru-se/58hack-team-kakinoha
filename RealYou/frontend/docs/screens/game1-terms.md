# Game 1: 利用規約ゲーム画面

**パス:** `/games/terms`

**前画面:** `/`（トップ） → **現在:** `/games/terms` → **次画面:** `/diagnosis`

**表示内容:**

- メインエリア: 縦に非常に長い（約5000px）スクロール可能な利用規約テキスト
- 固定フッター: テキスト入力欄（1行）、「同意する」ボタン、「同意しない」ボタン
- ポップアップ: 閲覧中に突然出現する広告モーダル（×ボタン付き）

**テキスト内の仕掛け（トラップ）:**

- 第5条付近: 「□ 読みました」チェックボックス
- 第8条付近: 「□ メルマガ受信」「□ 第三者提供」チェックボックス（初期値ON）
- 第12条付近: 「『確認済み』と入力してください」という指示文

**ユーザー行動フロー:**

1. 利用規約の長文をスクロールして読み進める（または読み飛ばす）
2. テキスト内のトラップに気づけば対応する（チェックを入れる/外す、指示通りに入力）
3. 突然出るポップアップ広告を閉じる
4. 固定フッターの「同意する」or「同意しない」ボタンを押して完了

**APIトリガー:**

- 「同意する」or「同意しない」ボタン押下時 → `POST /api/games/submit`
- 詳細なデータ定義は `docs/games/game1-terms.md` を参照

**完了時の動作:**

- 簡易診断画面（`/diagnosis`）へ自動遷移
