# Real You ~ 本当の私じゃだめですか？ ~
3つのミニゲームを通して、プレイヤーの「無意識の行動」を解析し、本質的な性格を暴き出す診断アプリケーションです。

## 🛠 前提条件
本プロジェクトをローカル環境で動作させるために必要なツールです。

- **Node.js**: v20.0.0 以上
- **npm**: v10.0.0 以上
- **Supabase**: データベースおよび認証基盤として使用
- **Gemini API**: 性格分析のAI生成に使用

## 🚀 起動方法
以下の手順でフロントエンドとバックエンドの両方を起動してください。

### 1. 依存関係のインストール
プロジェクトのルートディレクトリで以下のコマンドを実行します。
```bash
# バックエンド
cd backend && npm install

# フロントエンド
cd ../frontend && npm install
```

### 2. 開発サーバーの起動
それぞれのディレクトリで開発サーバーを起動します。

**バックエンド (Port: 3001)**
```bash
cd backend
npm run dev
```

**フロントエンド (Port: 3000)**
```bash
cd frontend
npm run dev
```

## 🔑 .envファイルの設定
それぞれのディレクトリに `.env` ファイルを作成し、必要なトークンやURLを設定してください。

### backend/.env
```text
PORT=3001
SUPABASE_URL=（SupabaseのプロジェクトURL）
SUPABASE_KEY=（SupabaseのAnon Key）
GEMINI_API_KEY=（Google AI Studioで発行したAPIキー）
FRONTEND_URL=http://localhost:3000
```

### frontend/.env.local
```text
NEXT_PUBLIC_API_URL=http://localhost:3001
```
