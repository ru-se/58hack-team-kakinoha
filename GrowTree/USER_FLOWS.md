# GrowTree ユーザーフロー

GrowTree は、GitHubの活動実績をもとにエンジニアとしての「ランク」を判定し、RPG風のスキルツリーとパーソナライズされた演習を提供するアプリです。

---

## 全体フロー概要

```
アクセス → 認証チェック → [未認証] ログイン → [初回] ランク測定 → ダッシュボード
                                                                    ↓
                                               スキルツリー閲覧 / 演習 / 成績確認
```

---

## 1. アクセス・認証フロー

### 1-1. トップページ (`/`)
1. ユーザーがサイトにアクセスする
2. 認証状態を自動チェック（ローディングスピナー表示）
3. **認証済み** → `/dashboard` へリダイレクト
4. **未認証** → `/login` へリダイレクト

---

## 2. ログイン・登録フロー (`/login`)

ログイン画面には **2つの方法** がある。

### 2-1. GitHub OAuth ログイン（推奨）
1. 「🚀 GitHub でログイン（推奨）」ボタンをクリック
2. GitHub の OAuth 認証ページへ遷移
3. GitHub 上で認証を承認する
4. バックエンドが GitHub リポジトリを分析し、**ランクを自動算出**
5. `/dashboard` へリダイレクト

### 2-2. ID / パスワードログイン（テスト用）
1. ユーザー名とパスワードを入力してフォームを送信
2. 認証成功 → `/dashboard` へリダイレクト
3. 認証失敗 → エラーメッセージを表示（シェイクアニメーション）

### 2-3. 新規登録
1. 「新規登録はこちら」をクリックしてフォームを切り替え
2. ユーザー名とパスワードを入力して送信
3. 登録・ログインが完了し、`/dashboard` へ遷移

---

## 3. ランク測定フロー (`/rank-measurement` またはダッシュボード上のモーダル)

**初回ダッシュボードアクセス時**に自動でモーダルとして表示される（以降は表示されない）。  
`/rank-measurement` から単独ページとしてもアクセス可能。

### 3-1. ランク測定の手順
1. **アイドル状態**: 画面に「🎯 ランクを測定」ボタンが表示される
2. **チャージ**: ボタンを**長押し**するとプログレスバーがチャージされる
   - 途中で離すとプログレスがゆっくり減少してリセット
   - チャージが 100% に達すると測定開始
3. **測定プロセス**（順番に進行するアニメーション）：
   - 📡 GitHub情報取得中（約2秒）
   - 🤖 AIが深層学習中（約3秒）
   - ⚙️ ランクを計算中（約2秒）
   - ✨ 最終確認中（約2秒）
4. **ランク確定（演出）**: 豪華なアニメーションとともにランクが発表される（約8秒）
   - ランク名（種子 → 芽 → 若木 → 樹 → 母樹 → 賢樹 → 神樹 → 世界樹）
   - 上位パーセンタイルの表示
   - AIの判定理由テキスト
   - バックグラウンドで全カテゴリのスキルツリーを生成開始
5. **完了**: 「🚀 ダッシュボードへ進む」ボタンが表示 → クリックでダッシュボードへ

---

## 4. ダッシュボードフロー (`/dashboard`)

### 4-1. 画面表示
1. データロード中はスピナーを表示（`LOADING USER DATA...`）
2. **初回アクセス時のみ**：ランク測定モーダルが全画面で表示される
3. ロード完了後、以下のセクションが表示される：
   - ウェルカムメッセージ（`WELCOME BACK, PLAYER`）
   - 前回の演習の「CONTINUE MISSION」ボタン
   - カテゴリセレクター
   - スキルツリー（インタラクティブなキャンバス）
   - 取得済みバッジ一覧

### 4-2. カテゴリ切り替え
1. カテゴリ（Web / AI / Security / Infrastructure / Game / Design）を選択
2. スキルツリーが選択カテゴリ用に再生成・再描画される
   - 生成中はプログレスバーで進捗を表示（`GENERATING SKILL TREE`）

### 4-3. スキルツリー操作
1. ドラッグでキャンバスをパン（移動）
2. スクロールでズームイン・ズームアウト
3. ノードをクリックするとサイドパネルに詳細情報を表示
4. ±ボタンでズーム操作、リセットボタンで初期位置に戻る

---

## 5. スキルツリー詳細ページ (`/skills`)

ダッシュボードとほぼ同じ操作が可能な、スキルツリー専用フルスクリーン画面。

1. スキルツリーキャンバス（ドラッグ・ズーム操作）
2. 画面右上に現在のランクバーを表示
3. 凡例（スキルレベルの色分け説明）を表示
4. ノードをクリックで詳細パネルを開閉

---

## 6. 演習フロー (`/exercises`)

### 6-1. 演習メニュー
演習ページは3つのタブで構成されている。

| タブ | 内容 |
|------|------|
| 🎓 通常演習 | カテゴリ別の既製演習一覧 |
| 📖 問題生成 | AIによる演習の自動生成 |
| 💻（開発中） | 現在準備中 |

### 6-2. 通常演習
1. カテゴリカード（Web / AI / Security / Infrastructure / Game / Design）をクリック
2. `/exercises/{カテゴリ}` へ遷移してレッスン一覧を表示
3. レッスンを選択すると `/exercises/{カテゴリ}/{exerciseId}` へ遷移
4. 演習コンテンツを閲覧・実施

### 6-3. AI問題生成（📖タブ）
1. テーマや条件を入力してAIに演習生成をリクエスト
2. AIがクエスト形式の演習を生成
3. 生成結果は `/exercises/generate/result` に表示：
   - クエストタイトル・概要
   - 学習目標
   - ステップ一覧（手順）
   - 参考リソース
4. 「🔄 もう一度生成する」または「🎮 演習一覧へ」ボタンで次のアクションへ

---

## 7. 成績確認フロー (`/grades`)

1. ユーザーの演習完了履歴・成績を一覧表示
2. 各演習のスコアや達成状況を確認できる

## ページ一覧・URL マップ

| URL | ページ | 認証 |
|-----|--------|------|
| `/` | トップ（認証チェック後リダイレクト） | 不要 |
| `/login` | ログイン・新規登録 | 不要 |
| `/dashboard` | ダッシュボード（スキルツリー・バッジ） | **必須** |
| `/skills` | スキルツリー専用ページ | **必須** |
| `/exercises` | 演習メニュー | **必須** |
| `/exercises/{category}` | カテゴリ別演習一覧 | **必須** |
| `/exercises/{category}/{id}` | 演習詳細 | **必須** |
| `/exercises/generate/result` | AI生成演習の結果 | **必須** |
| `/grades` | 成績一覧 | **必須** |
| `/rank-measurement` | ランク測定（単独ページ） | **必須** |
