# GrowTree ユーザーフロー(デモ版)

現在のフロントエンド実装（`/src/app/` 以下の構成）に基づいた実際のユーザーフローです。

---

## 全体フロー概要

```
アクセス ( / ) → [デモ版仕様] 強制的に /skills へリダイレクト
                                            ↓
               フルスクリーン スキルツリー閲覧・体験（デモ機能）
                                            ↓
---------------------------------------------------------------------------------

## 1. トップページ (`/`)

1. ユーザーが `http://localhost:3000/` にアクセスする。
2. ローディングスピナーが表示される。
3. **デモ版仕様**により、瞬時に `/skills` (スキルツリー専用デモページ) へ自動リダイレクトされる。（現在、ログインやダッシュボードを経由せずに基本機能を体験できるように設定されています。）

---

## 2. スキルツリー デモ体験 (`/skills`)

トップページからリダイレクトされる完全パブリックなスキルツリー画面です。

1. **キャンバス操作**: 画面全体に広がるスキルツリーをドラッグでパン移動、スクロールで拡大・縮小できる。
2. **ノード詳細確認**: ツリー上のスキルノードをクリックすると、画面下部に詳細パネルがポップアップする。
3. **デバッグ機能によるスキル解放テスト**:
   - 画面左下の「🛠 DEBUG」パネルを開く。
   - Web、AI、Securityなどのカテゴリごとにボタンを押し、擬似的に「ポイント」を付与する。
   - 未解放ノード（解放可能・AVAILABLEなもの）をクリックし、付与したポイントを消費して「スキル解放」ボタンを押すことで、スキルのアンロック・ツリーの進行をテストできる。
4. **階級・ステータス**: 画面左上のランクバー（EXPバーとバッジ）で、解放したノード数に応じた現在のツリー進行度・ランク（種子〜世界樹）を確認できる。
