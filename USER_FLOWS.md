# キメラアプリ企画 — ユーザー目線フロー

> 4つのアプリ（RealYou / TalkScope / TimeFaker / GrowTree）を統合するキメラアプリ企画のための、各アプリのユーザー体験まとめ。

---

## 🎮 RealYou（改造版）
**「空気読み診断 + 理解度チェック + 復習問題」キメラアプリ**

```
[1] トップ画面
    → 「スタート」ボタンを押す
         ↓
[2] 理解度チェック 説明画面 (/quiz)
    → 「これから理解度を答えてもらいます」と説明
    → タップで次へ
         ↓
[3] 理解度を5択で回答
    → 😵 全く理解できなかった
    → 😟 あまり理解できなかった
    → 😐 なんとなく理解できた
    → 😊 だいたい理解できた
    → 🤩 完全に理解できた
         ↓
[4] 復習タイム 説明画面
    → 「授業の振り返りをします」と説明
    → タップで次へ
         ↓
[5] 復習問題（1問ずつ）
    → 4択から回答を選ぶ
    → 即座に正誤フィードバック＋解説を表示
    → 全問終了まで繰り返す
    （問題はquizContent.tsで一括編集可能）
         ↓
[6] 結果レポート (/result)
    → 空気読みスコアを表示（予定）
```

**シナリオ編集ファイル:**
- `frontend/src/features/games/group-chat/data/scenario.ts` — グループチャットの内容
- `frontend/src/features/quiz/data/quizContent.ts` — 理解度・復習問題の内容

## 🎙️ TalkScope
**「難しい会議・講義でリアルタイムに専門用語の意味を確認できる」**

```
[1] ブラウザを開く（ログイン不要）
         ↓
[2] 「録音開始」ボタンを押す
    → マイクで音声をキャプチャ開始
         ↓
[3] 会話・講義が進む
    → リアルタイムで音声が文字起こしされる（Web Speech API）
    → バックエンドが専門用語を自動抽出・スコアリング
         ↓
[4] 専門用語が強調表示される
    → テキスト内でハイライト
    → 重要度に応じた大きさのバブル（泡）が画面に浮かぶ
         ↓
[5] 気になる単語を操作
    → ホバー   → 簡単な説明がポップアップ
    → クリック → 詳細説明を表示
    → 右クリック → ピン留め（後で見返せる）
         ↓
[6] 録音終了後
    → ピン留めした単語一覧を確認・再参照
    → IndexedDBに保存されるのでリロード後も残る
```

## ⏰ TimeFaker
**「チームメンバーの時計を密かにズラして遅刻癖を治す（？）いたずらシステム」**

### 管理者（仕掛ける側）のフロー
```
[1] Discord で Bot に Gmail を登録
    → !register メールアドレス
         ↓
[2] Google カレンダーを連携
    → !auth コマンド → OAuth URLが届く → ブラウザで認証
         ↓
[3] 予定を登録
    → !plan 2026-03-01 10:00 ミーティング
    → メンションした相手にも攻撃予約が入る
         ↓
[4] 自動または手動で発動
    → 午前0時に自動発動（Googleカレンダーに予定がある人だけ対象）
    → または物理ボタン（ラズパイ）のシングルクリックで即時発動
         ↓
[5] ターゲットのデバイスが30〜120分進む
    → 📱 スマホ：MDM経由でタイムゾーン変更
    → 💻 Windows PC：WebSocket経由で時刻変更
    → ⌚ 物理時計：BLEビーコンで変更
         ↓
[6] 復旧
    → 物理ボタン長押し（4秒）で全デバイスを正常時刻に戻す
```

### 被害者（やられる側）のフロー
```
[1] 何もしていない
         ↓
[2] 気づいたら時計が30〜120分進んでいる
         ↓
[3] 「え、もうこんな時間！？」と慌てて行動する
```
