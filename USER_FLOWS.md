# キメラアプリ企画 — ユーザー目線フロー

> 4つのアプリ（RealYou / TalkScope / TimeFaker / GrowTree）を統合するキメラアプリ企画のための、各アプリのユーザー体験まとめ。

---

## 🎮 RealYou
**「無意識の行動から、本当の自分の性格を診断する」**

```
[1] トップ画面
    → 「スタート」ボタンを押す
         ↓
[2] ゲーム1「規約の罠」
    → 普通の利用規約に見える画面が表示される
    → 読んだかどうか、不要なチェックを外すかどうかを「こっそり」計測
    → しばらくするとポップアップ広告も登場（反応を観察）
    → 「同意する」「同意しない」いずれかを選択して次へ
         ↓
[3] MBTI自己申告（主観）
    → 「あなたはどんな性格？」を自分で回答
         ↓
[4] ゲーム2「AIヘルプデスク」
    → ポンコツAIに悩みを相談する（音声 or テキスト入力）
    → AIの理不尽な回答に対する反応（タイピングの乱れ・修正・速度）を計測
         ↓
[5] ゲーム3「空気読み」
    → LINEグループ風のチャットに参加
    → ボットが意見を出し合い、多数派が形成されていく
    → 選択肢から返信を選ぶ（同調するか、独自意見を貫くか）
    → 選ぶまでの迷い時間も計測
         ↓
[6] 結果レポート
    → 「自己申告の性格（主観）」vs「行動データから測定した性格（客観）」
    → レーダーチャート＋バーチャートでギャップを可視化
    → 慎重さ / 冷静さ / 論理性 / 協調性 / 積極性 の5軸で評価
```

## 🎙️ TalkScope
**「難しい会議・講義でリアルタイムに専門用語の意味を確認できる」**

```
[1] ブラウザを開く（ログイン不要）
         ↓
[2] 「録音開始」ボタンを押す
    → マイクで音声をキャプチャ開始
         ↓
[3] 会話・講義が進む
    → リアルタイムで音声が文字起こしされる（Web Speech API）
    → バックエンドが専門用語を自動抽出・スコアリング
         ↓
[4] 専門用語が強調表示される
    → テキスト内でハイライト
    → 重要度に応じた大きさのバブル（泡）が画面に浮かぶ
         ↓
[5] 気になる単語を操作
    → ホバー   → 簡単な説明がポップアップ
    → クリック → 詳細説明を表示
    → 右クリック → ピン留め（後で見返せる）
         ↓
[6] 録音終了後
    → ピン留めした単語一覧を確認・再参照
    → IndexedDBに保存されるのでリロード後も残る
```

## ⏰ TimeFaker
**「チームメンバーの時計を密かにズラして遅刻癖を治す（？）いたずらシステム」**

### 管理者（仕掛ける側）のフロー
```
[1] Discord で Bot に Gmail を登録
    → !register メールアドレス
         ↓
[2] Google カレンダーを連携
    → !auth コマンド → OAuth URLが届く → ブラウザで認証
         ↓
[3] 予定を登録
    → !plan 2026-03-01 10:00 ミーティング
    → メンションした相手にも攻撃予約が入る
         ↓
[4] 自動または手動で発動
    → 午前0時に自動発動（Googleカレンダーに予定がある人だけ対象）
    → または物理ボタン（ラズパイ）のシングルクリックで即時発動
         ↓
[5] ターゲットのデバイスが30〜120分進む
    → 📱 スマホ：MDM経由でタイムゾーン変更
    → 💻 Windows PC：WebSocket経由で時刻変更
    → ⌚ 物理時計：BLEビーコンで変更
         ↓
[6] 復旧
    → 物理ボタン長押し（4秒）で全デバイスを正常時刻に戻す
```

### 被害者（やられる側）のフロー
```
[1] 何もしていない
         ↓
[2] 気づいたら時計が30〜120分進んでいる
         ↓
[3] 「え、もうこんな時間！？」と慌てて行動する
```
